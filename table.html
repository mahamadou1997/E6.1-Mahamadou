Atributo Descripción Explicación Ejemplo
background-color Establece el color de fondo de un elemento La propiedad background-color se emplea para establecer el color de fondo de un elemento mediante cualquiera de las diferentes formas de indicar un color en CSS div { background-color: #E0ECFF; } Esto pone el fondo de color azul
background-image Establece la imagen de fondo de un elemento La propiedad background-image se emplea para establecer la imagen de fondo de cada elemento. La imagen se muestra en la zona que ocupan el contenido y el relleno del elemento, justo hasta su borde. div { background-image: url: De la imagen.gif); }
background-repeat Controla el tipo de repetición de las imágenes de fondo La propiedad background-repeat controla la forma en la que se repiten las imágenes de fondo de los elementos. div { background-image: url("images.png"); }
font-size Establece el tamaño de letra. La propiedad font-size permite establecer el tamaño de letra del texto. Además de todas las unidades de medida absolutas y relativas conocidas (em, ex, px, %, in, cm, mm, pt, pc) también se pueden utilizar palabras clave que hacen referencia a tamaños de letra absolutos y relativos. font-size: xx-large
font-weight Establece el grosor de cada letra La propiedad font-weight permite establecer el grosor con el que se muestran las letras del texto. Normalmente, la propiedad font-weight se emplea para mostrar un texto en negrita (valor bold): style="font-weight: bold;"
font-family Establece la fuente o tipo de letra La propiedad font-family establece la fuente o tipo de letra con la que se muestra el texto. Cuando no se establece de forma explícita el tipo de letra, los navegadores muestran el texto con un tipo de letra por defecto. "font-family 'Trebuchet MS';">
text-align Establece la alineación del contenido de un elemento de bloque text-align no sólo controla la alineación del texto. En realidad, esta propiedad establece la alineación de los contenidos (texto, imágenes) que se encuentran dentro de un elemento de bloque. text-align: justify, center, left, right
text-transform Transforma el texto de un elemento para mostrarlo en mayúsculas, minúsculas o una mezcla de ambas La propiedad text-transform permite transformar el texto de un elemento para mostrarlo en mayúsculas, minúsculas o una mezcla de ambas. Los cuatro valores definidos por esta propiedad son los siguientes: text-transform: none text-transform: lowercase
border Establece algunas o todas las propiedades de todos los bordes de los elementos La propiedad border es una de las "propiedades shorthand" que define CSS y que se utilizan para establecer de forma abreviada el valor de una o más propiedades individuales. div { border: 1px solid #C00; }
list-style-type Establece el marcador de los elementos de la lista La propiedad list-style-type establece el marcador que muestra cada elemento de las listas HTML. Por defecto, los navegadores muestran unos pequeños círculos rellenos para cada elemento de las listas no ordenadas y una numeración decimal para los elementos de las listas ordenadas

padding Establece la anchura de algunas o todas las zonas de relleno de los elementos La propiedad padding es una de las "propiedades shorthand" que define CSS y que se utilizan para establecer de forma abreviada el valor de una o más propiedades individuales. 0div { padding-top: 10px; padding-right: 10px; padding-bottom: 10px; padding-left: 10px; }
margin Establece la anchura de algunos o todos los márgenes de los elementos La propiedad margin es una de las "propiedades shorthand" que define CSS y que se utilizan para establecer de forma abreviada el valor de una o más propiedades individuales. div { margin-top: 10px; margin-right: 10px; margin-bottom: 10px; margin-left: 10px; }
